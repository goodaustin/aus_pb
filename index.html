2

3
Cloud semis 沒有降速，CPU/general server 反而成為 AI capex 的低調槓桿：MS 在 WAIC 看到 AMD / NVIDIA / 海光方案拉動 CPU server，信驊 BMC 2030 TAM 從 46.5mn 上調到 65.77mn units、AST2700 ASP +40-50%；GPU server 2027 可能更多 SuperPods / multi-rack scale-up，和今天 CPO optical scale-up memo 互相咬合。
