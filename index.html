菲想早報.
2026-07-20（一）盤前
今日三件事
1
AI infra 的受惠範圍繼續往外擴，不只 GPU / ODM：緯創吃 NVL72、VR200、Vera CPU rack；邁科吃 ASIC 液冷滲透；順達吃 BBU；台灣 CCL 吃 M7+ / M8+ shortage；研華吃 edge AI / automation；這批報告共同指向同一件事：AI capex 已從 compute board 擴散到 rack thermal、power backup、high-end material、industrial edge device。
2
高階 CCL 是今天最硬的供需故事：GS 把 2026/27 CCL TAM 估到 US$20.0bn / 34.1bn，2028 到 US$57.0bn，總 CCL 2025-28E CAGR 53%，其中 high-end CCL 96% CAGR，供給只 22% CAGR；M7+ 2H26 可能再漲 10-15%，EMC / TUC TP 大幅上修。這不是成本推升，是 AI PCB bottleneck pricing power。
3
研華兩家券商都上修，但結論分裂：UBS Buy、TP NT$628，認為 2026/27 EPS 高於 consensus 22-29%；BofA Neutral、PO NT$515，承認 2Q sales / EPS beat 但說估值已 fair。差異核心不是需求，而是能否持續 pass-through memory / CPU / PCB 成本，並把 edge AI design-win 轉成 2027 earnings。
