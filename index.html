2026-07-17（五）盤前
今日三件事
1
台積電法說後，AI supply chain 的估值焦點從「有沒有需求」轉到「哪個 bottleneck 會被 capex 砸穿」：台積電把 2026 capex 拉到 US$60-64bn、全年美元營收成長略高於 40%，等於確認 advanced node / advanced packaging / test / substrate / power / cooling 都不是短線拉貨。這批報告剛好把下一層瓶頸補齊：Ibiden 是 ABF / EMIB-T / GPU substrate，ASMPT 是 TCB / CoWoS / HBM，南茂是 SiPh bumping + Logic/ASIC test，晶技是 AI optical XO，MS scale-up primer 則把 copper / CPO / testing 的 2028-2030 路線攤開。
2
Scale-up networking 是新主線，不是 CPO 單點題材：MS 把 scale-up backend network TAM 2030 拉到 US$73bn，較去年估計放大 4 倍。結論很關鍵：copper 會活得比市場想像久，因 latency / power / cost / ecosystem 仍有優勢；但 multi-rack scale-up、I/O power、bandwidth density 會逼 optical / CPO 於 2029 後更明確放量。這對台股不是只看光通訊，而是同時看智邦/交換器、貿聯/銅互連與 shuffle box、晶技/XO、南茂/SiPh bump、穎崴/旺矽/測試，以及 Keysight 類測試設備 read-through。
3
這批台股新料最可交易的不是單一大報告，而是「AI 基建外溢到小零件/成熟工藝」：南茂 memory 復甦 + 3Q25 起 5-18% 漲價 + 2H26 SiPh trial run + 2026 底 Logic test 交機；晶技 7 月漲價 10-30%，AI/光通訊與車用是毛利率主軸；Acer 2Q sales beat 但 BofA 仍 Underperform，提醒 PC 品牌可漲價但 margin 被 memory/component cost 吃掉。日本/港股則補強海外供應鏈定位：Ibiden ABF/EMIB-T、ASMPT TCB、Nippon Chemi-Con AI server extended-length capacitor、Huaqin 中國 rack-level AI ODM。
